Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tab B (Não semantica)

Nessa frase temos um Termo em destaque usando a tag Strong (Semantica)

Itálico / Ênfase

Nesta frase, temos um termo em Italico usando a tag I (Não semantica)

Nesta frase, temos um termo em ênfase usando a tag EM (Semantica)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK (Semantica).

Podendo alterar a cor da marcação com o comando: STYLE="BACKGROUND-COLOR : COLORAÇÃO;"
Totalmente em minusculo. Comando que é realizado pelo CSS

Texto Grande / Pequeno

Estamos criando um Texto grande e um Texto pequeno neste paragrafo

Usando simultaneamente a tag BIG e a tag SMALL
A tag BIG é errado usar, já a SMALL é correta

Texto Deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido, mas não considerado.
Com a tag DEL (Semantica)

Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois
Com a tag INS (Semantica)

Existe também o sublinhado com a tag U (NÃO Semantica)

Texto Sobrescrito

Para inserir numeros deste tipo : x20+3

Com a tag SUP (Semantica)

Texto Subscrito

Para inserir numeros deste tipo: H2O
aaa
