Reproduzindo Audio

Vamos aprender a reproduzir um audio em HTML5

        <audio src="midia/timeless.mp3" controls></audio>

Forma 2

        <audio preload="metadata" controls>
            <source src="midia/on_the_island.mp3" type="audio/mpeg">
            <source src="midia/timeless.mp3" type="audio/mpeg">
            <p>Seu navegador não tem compatibilidade com audios. <br><a href="midia/timeless.mp3">Clique aqui para fazer o Dowload do audio</a></p>
        </audio>
    
Esta forma é usada para caso 1 audio não funcione o navegador ira abrir o segundo, 
e caso também não funcione ira mostrar a mensagem e o link!

Parametros

        <audio controls>
        Utiliza para mostrar uma controladora do audio 
        

        <audio loop>
        Utiliza para definir um loop no audio
        

        <audio preload="metadata">
        Utiliza para o navegador apenas fazer o dowload do audio no momento em que ele for iniciado
