Como hospedar vídeos externos

Para fins de economia com hospedagem em servidores,
Utiliza-se a hospedagem externa, geralmente através do Youtube!

O processor para pegar o vídeo hospedado e colocar no site é um passo a passo

  1. Encontre o vídeo que vai colocar no site
  2. Ao abrir o vídeo, clique em compartilhar
  3. Clique no icone incorporar
  4. Copie, e cole no código
