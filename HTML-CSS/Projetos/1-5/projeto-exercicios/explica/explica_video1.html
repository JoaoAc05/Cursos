Como Colocar Vídeos

Para esta tarefa foi criado uma pagina pois não se trata apenas de uma tag, e sim de um grupo de comandos e parametros!
O código é muito semelhante a aplicação de audio
Mudando apenas a Tag e alguns parametros 

    Tendo como sua forma simples e sem adaptação para compatibilidade

    <video src="../../ex012/midia/meu_video.webm" controls></video>

    com apenas uma extensão sendo exibida!

Por questões de maior compatibilidade e design podemos fazer 
        
        <video width="400" controls poster="../../ex012/imagem/tumb.png">
            <source src="../../ex012/midia/meu_video.webm" type="video/webm">
            <source src="../../ex012/midia/meu_video.mkv" type="video/mkv">
            <source src="../../ex012/midia/meu_video.mp4" type="video/mp4">
            <source src="../../ex012/midia/meu_video.ogv" type="video/ogg">
            <p>Seu navegador não tem compatibilidade com a apresentação de vídeo</p>
        </video>   
        


    
Parametros

    
    
        
  1. width
    2. 
Utilizado para determinar o tamanho que ira aparecer o vídeo   
        
  2. controls
    3. 
Utilizado para poder iniciar e pausar o vídeo
        
  3. poster
    4. 
Utilizado para definir uma imagem inicial do video
        
  4. source
    5. 
utilizado para caso o navegador não tenha compatibilidade com a extensão do vídeo
ele ira abrir a próxima extensão e caso não tenha compatibilidade também 
ira escrever o texto de aviso
        
  5. loop
    6. 
define um loop no video quando ele acabar
